# stylelint-config-cy

> stylelint config for cy

## Install

```sh
npm install prettier stylelint @cyys/stylelint-config-vue --save-dev
```

If using Yarn:

```sh
yarn add prettier stylelint @cyys/stylelint-config-vue --dev
```

## Usage

.stylelintrc.json

```json
{
  "extends": "@cyys/stylelint-config-vue"
}
```
